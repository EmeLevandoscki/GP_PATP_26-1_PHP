Como a inscrição funciona

O organizador define quais dados serão solicitados em cada evento. A página pública exibe apenas os campos configurados, evitando formulário pesado e genérico.

01

Organizador cria o evento

Define título, descrição, data, vagas, categoria, imagem e publicação.

02

Configura os campos

Escolhe CPF, e-mail, telefone, curso, comunidade e campos extras.

03

Divulga o link

Cada evento possui link individual, como evento.html?id=....

04

Acompanha inscritos

O painel mostra totais, ocupação, inscritos recentes e exportação CSV.
